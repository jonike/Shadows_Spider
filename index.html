Shadow's Spider

Cross Platform OpenGL 4.5 Realtime Graphics

Download as .zip Download as .tar.gz View on GitHub

Shadow's Spider is a graphics program that's fun to tweak and optimize.

Features

  • Manipulate objects in multiple ways
    • add / delete / duplicate
    • translate / rotate / scale with gizmos or AE
  • Quick workflow
    • jump between and look through lights
    • radial menus
  • PBR shaders
    • IBL with pre-convolved cubemaps, oren-nayar, ggx
    • adaptive luminance, bloom, fxaa, ssao, tonemapping
  • Qt GUI
  • Linux and Windows
    • C++ and GLSL
  • GPLv3
