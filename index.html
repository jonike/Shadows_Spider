Shadow's Spider

Cross Platform OpenGL 4.5 Realtime Graphics

Download as .zip Download as .tar.gz View on GitHub

Features

  • Manipulate objects in multiple ways

    • add / delete
    • translate / rotate / scale with gizmos or AE

  • Quick workflow

    • look through lights
    • radial menus

  • PBR-ish IBL shaders

    • easy to follow shaders that work with multiple lights
    • adaptive luminance, bloom, tonemapping, ssao, fxaa in HDR postprocessing chain

  • Qt GUI

    • tear-off, drag, resize, and reposition windows
    • easy windowing system
