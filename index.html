Shadow's Spider

Cross Platform OpenGL 4.5 Realtime Graphics

Download as .zip Download as .tar.gz View on GitHub

Shadow's Spider is a graphics program that fits my needs. Don't get me wrong - I like existing programs. But, I want to tweak 'em.

Features

  • Manipulate objects in multiple ways
    • add / delete / duplicate
    • translate / rotate / scale with gizmos or AE
  • Quick workflow
    • jump between and look through lights
    • radial menus
  • PBR shaders
    • IBL with pre-convolved cubemaps, oren-nayar, ggx
    • adaptive luminance, bloom, fxaa, ssao, tonemapping
  • Qt GUI
  • Linux and Windows
    • Easy to follow C++
  • GPLv3
